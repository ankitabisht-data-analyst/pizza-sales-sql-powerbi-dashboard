# 📁 SQL Folder

| File | Description |
|---|---|
| `pizza_sales_queries.sql` | 15 MySQL queries covering total revenue, AOV, total pizzas sold, daily/monthly order trends, % of sales by category and size, and top/bottom 5 pizzas by revenue, quantity, and order count. |

**How to use:** Create a database named `pizza_db`, load `pizza_sales.csv` from `/data` into a `pizza_sales` table, then run these queries directly or use them as the basis for Power BI DAX measures.
